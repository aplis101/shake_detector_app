# Shake Detector App 📱👋

Project sederhana ini saya bangun untuk memenuhi tugas Praktikum Pemrograman Mobile. Aplikasi ini memanfaatkan sensor **Accelerometer** bawaan smartphone untuk mendeteksi guncangan (shake).

### Cara Kerjanya gimana?
Sangat simpel! 
- Saat HP diguncangkan dengan kuat ke arah mana saja (kiri/kanan/atas/bawah), sensor akan merespons dan mengubah warna background layar menjadi merah.
- Saat posisi HP kembali stabil atau diam, layar akan otomatis kembali ke warna hijau.

### Tech Stack 🛠️
- **Framework:** Flutter
- **Library:** `sensors_plus`

Dibuat murni untuk keperluan eksplorasi sensor mobile dan penyelesaian tugas kampus.
